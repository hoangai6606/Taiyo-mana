Taiyo Nisshin Vietnam - Project Handoff for Coding Agent

 Tổng quan Project

 Hệ thống quản lý Kiểm hàng (QC Inspection Management) cho công ty may
 mặc Taiyo Nisshin Vietnam. Thay thế quy trình quản lý bằng Excel bằng
 web app, bao gồm:
 - Theo dõi báo cáo kiểm hàng hàng ngày (defect tracking theo
 product/color/size)
 - Theo dõi năng suất sản xuất
 - Tạo debit note tự động từ inspection reports đã duyệt
 - Export Excel theo template khách hàng
 - Phân quyền theo role (staff, leader, manager, accounting_admin,
 super_admin)
 - Hỗ trợ multi-tenant workspace

 Database Schema (18 tables)

 Core

 - workspaces - Multi-tenant workspace
 - profiles - Users (email, password_hash, name_vn, name_jp, role,
 factory_id, workspace_id, active)
 - user_factory_permissions - User-factory access (full_access,
 read_write, read_only)

 Master Data

 - customers - Khách hàng (code, name, currency USD/JPY)
 - factories - Nhà máy (code, name, country)
 - product_types - Loại sản phẩm
 - product_styles - Mã hàng (code, customer_id, factory_id)
 - factory_prices - Giá theo nhà máy

 Inspection

 - inspection_records - Phiếu kiểm hàng (date, factory, customer,
 product, order_qty, status)
 - inspection_items - Chi tiết kiểm hàng (color, size, inspected_qty,
 passed_qty, defects...)
 - inspection_reports - Báo cáo kiểm hàng (generated from records)
 - report_items - Items trong report
 - report_productivity - Năng suất
 - daily_reports - Báo cáo ngày
 - productivity_tracking - Theo dõi năng suất (date, qc_quantity, ot)

 Finance

 - debit_notes - Phiếu debit (customer, factory, period, total)
 - debit_note_items - Chi tiết debit (product, qty, unit_price,
 line_total, item_type)

 Chat

 - chat_messages - Tin nhắn (workspace_id, sender_id, message, file_url)

 ---
 Auth & Roles

 Roles: staff, leader, manager, accounting_admin, super_admin

 Flow:
 1. Login → JWT token → stored in localStorage
 2. Mọi API call đều kèm Authorization: Bearer <token>
 3. Middleware authenticateToken verify JWT
 4. Super admin có thể impersonate workspace khác

 Navigation theo role:
 - Super admin: Dashboard + Admin + tất cả pages
 - Manager/Leader/Staff: Inspection + Reports + Debit Notes (theo
 factory permission)

 ---
 Business Rules quan trọng

 - Inspection workflow: draft → submitted → approved → locked
 - Locked reports không edit được (trừ manager có audit log)
 - Debit notes chỉ tạo từ approved reports
 - Default defect threshold: 15%
 - Excel export phải khớp chính xác template khách hàng
 - Tất cả master data dùng soft delete (is_active flag)
